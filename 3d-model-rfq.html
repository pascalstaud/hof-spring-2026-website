Request for Quotation

3D Base Model — Mercedes-AMG G-Class (W465)
HOF Manufaktur GmbH · Confidential · March 2026

1. Project Overview

HOF requires a production-ready 3D base model of the Mercedes-AMG G63 (W465) in Blender, built from CAD data provided by HOF. The model must support multiple geometric variants, configurable components, and studio-quality rendering for website imagery, customer configurator visualisations, and AI-assisted image generation.

Deliverable: One Blender file (.blend) containing the complete, fully configurable 3D model with all variants, materials, and grouped elements — returned to HOF upon project completion.

2. Scope of Work — Quotation Positions

Pos. Description Type
1 3D Base Model Setup
Import and preparation of CAD data (IGS/Step format) in Blender. Clean-up, topology optimisation, and establishment of the base 5-door G-Class model as the working foundation. 		Core
2 Two Exterior Silhouette Variants
Salon Silhouette — standard G-Class proportions, clean elegant surfaces.
Sport Silhouette — widened fenders, sculpted surfaces, modified bonnet, different front/rear bumper geometry.
Both silhouettes must be switchable within the same Blender file. 		Core
3 Three Door/Body Configurations
Each silhouette must support three configurations:
• Regular — factory bumpers, standard front-hinged doors
• Coach Doors — reverse-opening (suicide) rear doors, standard bumpers
• Cleaned & Coach Doors — side bumpers/cladding removed + Coach Doors
Switchable via grouped elements. 		Core
4 Interior Soft-Part Modelling
Complete interior modelling including:
• Standard rear bench seat
• Lounge Seats (individual rear seats replacing bench — new geometry, not in CAD)
• Entry sill plates / door step panels (new geometry)
• Dashboard, centre console, door panels, headliner
All interior surfaces must accept material/texture assignments. 		Core
5 Configurable Grouped Elements
All exterior and interior components must be grouped and individually colour/material-assignable:
• Exterior: bonnet, doors, fenders, bumpers, side cladding, roof, spare wheel cover
• Interior: seats (front/rear), dashboard, console, door cards, headliner, carpet, steering wheel
• Wheels: multiple rim designs, switchable
• Steering wheel: standard AMG + F1-style variant 		Core
6 Material & Texture Setup
Application of HOF material library to all surfaces. Textures provided by HOF where available (leather, textile, carpet). Remaining materials to be created/matched from reference imagery. 		Core
7 Studio Rendering Environment
Clean studio setup in Blender for production-quality renders:
• Neutral light grey/white cyclorama background
• Three-point studio lighting, warm neutral tone
• Subtle ground reflection
• Minimum resolution: 4000 × 2600 px
• Output: PNG (transparent bg) + JPEG (studio bg) 		Core
8 Model Handover
Delivery of the final Blender file (.blend) with all variants, materials, and scene setup to HOF for continued internal use. Includes brief documentation of file structure and element grouping. 		Core
9 Optional: 3-Door Variant
Extension of the base model to include the 3-door G-Class body variant. Separate geometric adaptation of all relevant panels, doors, and interior. 		Optional
10 Optional: Cabriolet Variant
Extension of the base model to include the G-Class Cabriolet body variant. Roof mechanism, windscreen frame, roll bars, modified rear section. 		Optional

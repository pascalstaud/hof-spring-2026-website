Technical Requirements

  • Resolution: Minimum 4000 × 2600 px (landscape). Retina-ready for web.
  • Format: PNG with transparent background (preferred) + JPEG on studio background.
  • Studio Background: Clean light grey/white cyclorama. Soft, even lighting. Subtle ground reflection.
  • Lighting: Three-point studio lighting. Warm neutral tone. No harsh shadows. Soft gradient on floor.
  • Base Vehicle: Mercedes-AMG G63 W465. All models share the same platform architecture.
  • Silhouette Types: Salon Silhouette = standard G-Class proportions, pure/clean surfaces. Sport Silhouette = widened fenders, sculpted surfaces, aggressive stance.
  • Coach Doors: Only Kinsō has Coach Doors (reverse-opening rear-hinged doors). All other models use standard front-hinged doors.
  • Lounge Seats: Only Kinsō has individual S-Class rear seats replacing the bench. Tokyo, Notan, Ma use standard rear bench.
  • File Naming: Use the image IDs from this document (e.g., K-I-P1.png, T-II-P5.png, M-I-D2.png).
  • Color Accuracy: Color hex values are indicative. Final color matching from physical samples to be provided separately.
